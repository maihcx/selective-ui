# Selective UI
Advanced Custom Select Component - Pure JavaScript, Zero Dependencies

**Selective UI** is a **DOM-driven select component** where the native `<select>` remains the single source of truth

Full docs with examples: http://selective-ui.maisoft.io.vn/

## Usage

```ts
registerPlugin(MyPlugin);
bind(".my-select");
```
